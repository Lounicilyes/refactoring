## À propos de l’historique Git

Vous pourrez remarquer que l’historique Git de ce dépôt est un peu particulier.

Au départ, j’ai développé ce projet dans un autre dépôt, `tdd-tp`.  
Je me suis ensuite rendu compte que ce n’était pas le bon dépôt pour ce TP de refactoring.

Pour corriger ça :

- j’ai d’abord travaillé et commité dans le dépôt `tdd-tp`,
- puis j’ai créé ce nouveau dépôt `refact-tp`,
- et j’ai poussé ici tout le travail (commits, branches, etc.).

C’est pour cela que :

- certains commits ont des dates récentes alors que le développement avait commencé plus tôt,
- et que l’historique peut faire référence à l’ancien dépôt `tdd-tp` ou à d’autres TPs.

L’**état actuel du code** dans ce dépôt correspond bien au projet de refactoring demandé.
